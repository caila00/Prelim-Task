    Block Element

    A block element always starts on a new line and takes up the full width available (from left to right). It also pushes the next element to a new line.

    examples: <h1> , <h6>, <ul>, <ol>, <li>, <table>

    Inline Element

    An inline element does not start on a new line. It only takes up the space that its content needs.

    examples: <span>, <a>, <img>, <strong>, <em>
